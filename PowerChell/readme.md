# PowerChell Usage Notes

This repository contains compiled binaries and usage notes for PowerChell.

For educational and lab use only.

---

## Usage

### DLL Version

Execute the DLL using `rundll32`:

    rundll32 PowerChell.dll,Start

If using a full path:

    rundll32 C:\Path\To\PowerChell.dll,Start

---

### EXE Version

Run the executable directly:

    PowerChell.exe

Or with full path:

    C:\Path\To\PowerChell.exe
